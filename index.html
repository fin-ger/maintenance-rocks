Maintenance Rocks!

Reaching out to the open source community to help maintain your project.

You are a developer of a small open source project and don't know how to properly maintain it or simply do not have the time to do it?
Ask other developers for help!
How? Add the badge to your project and encourage other developers to get in touch with you to help you out!
Adding the badge to your project
First, you need to tell other developers how to contact you. The easiest way to do this is by adding a CONTACT.md file to your project. This file might look like this: 
# Contact

If you encounter any problems with the software please open an issue in the
issue tracker of this project.

If you are looking for a way to get in touch with the developer choose one
of the following methods:

 - [Telegram](t.me/username)
 - [Email](mailto:my@email.address)
Next you can add one of the following snippets to your projects README.md and replace my.contact.url with your contact URL:
Markdown 
[![Looking for Maintenance](https://maintenance.rocks/badge.svg)](https://offering.maintenance.rocks#my.contact.url)
reStructuredText 
.. image:: https://maintenance.rocks/badge.svg
    :alt: Looking for Maintenance
    :target: https://offering.maintenance.rocks#my.contact.url
Textile 
!https://maintenance.rocks/badge.svg!:https://offering.maintenance.rocks#my.contact.url
Rdoc 
{<img src="https://maintenance.rocks/badge.svg" alt="Looking for Maintenance" />}[https://offering.maintenance.rocks#my.contact.url]
AsciiDoc 
image:https://maintenance.rocks/badge.svg["Looking for Maintenance", link="https://offering.maintenance.rocks#my.contact.url"]
HTML 
<a href="https://offering.maintenance.rocks#my.contact.url"
   alt="Looking for Maintenance">
    <img src="https://maintenance.rocks/badge.svg">
</a>
And you are done! Congratulations, you have successfully added the badge to your project. Now sit back and see if someone is willing to help you out.
